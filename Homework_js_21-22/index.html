Инструкция к ДЗ 21-22 (часть 1).

1. Клонировать или обновить локальный репозиторий a-shaforostov.github.io
git@github.com:a-shaforostov/a-shaforostov.github.io.git

2. Открыть консоль в папке Homework_js_21-22
cd Homework_js_21-22

3. Установить зависимости
npm install

4. Запустить тест
npm test

5. Выполнить функцию из консоли
node js\script.js <base> <power>
base - Основание степени
power - Степень
Например: node js\script.js 2 3
